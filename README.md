# 🛡️ Surjob AI — #JuaraVibeCoding

> **Surjob AI** adalah solusi cerdas berbasis AI yang dirancang khusus untuk memberdayakan pekerja di Indonesia dalam menghadapi tantangan di tempat kerja. Proyek ini dikembangkan sepenuhnya dengan pendekatan **VibeCoding** — mengubah ide kreatif menjadi aplikasi nyata menggunakan bahasa alami.

Aplikasi ini dibangun dengan ekosistem **Google Cloud** untuk performa, skalabilitas, dan keamanan:

| Layer | Teknologi | Fungsi |
|---|---|---|
| 🤖 AI Engine | Google Gemini (via AI Studio) | Otak di balik analisis situasi kerja & generator solusi legal |
| 🎨 Frontend | React + Vite + Tailwind CSS | Tampilan modern, responsif, dan interaktif |
| ⚙️ Backend | Node.js + Express.js | Logika server-side dan routing API |
| 🗄️ Database | Firebase Firestore | Data komunitas & leaderboard secara *real-time* |
| ☁️ Deployment | Google Cloud Run | *Auto-scaling*, selalu *live*, infrastruktur terkelola |
| 🔨 Vibe Method | Google Gemini | Seluruh logika disusun via kolaborasi intensif dengan AI |

## 💡 Project Vision

Memberdayakan pekerja Indonesia untuk memahami hak-hak ketenagakerjaan mereka, melakukan analisis situasi kerja secara mandiri, dan mendapatkan dukungan legal serta emosional dalam proses yang **100% anonim dan profesional**.

---

## 🎯 Target Audience

Pekerja di seluruh sektor industri di Indonesia yang memerlukan dukungan saat menghadapi:

- 😤 Lingkungan kerja *toxic* atau *gaslighting*
- 📄 Pelanggaran kontrak kerja dan pengupahan
- 🚪 Kebingungan mengenai prosedur pengunduran diri (*resign*) yang aman

---

## ✨ Fitur Utama

- **🤖 Konsultasi AI** — Ceritakan masalah kerja, AI merespons dengan empati dan analisis hukum
- **📊 Skor Kesehatan Kerja** — Rangkuman otomatis berupa skor dan red flag dari percakapan
- **📝 Generator Surat** — Draf surat somasi dan kuasa kerja secara instan
- **🏆 Leaderboard Solidaritas** — Komunitas anonim pekerja yang saling menguatkan
- **🔒 100% Anonim** — Tidak ada data identitas yang disimpan atau dilacak

---

## 🛡 Security & Privacy

Kami menerapkan praktik keamanan terbaik untuk melindungi data pengguna:

- **Environment Management** — Kredensial sensitif dikelola via *Environment Variables* di Google Cloud Run
- **Database Security** — *Firestore Security Rules* memvalidasi setiap akses data
- **Privacy First** — Identitas pengguna disamarkan sejak awal interaksi, tidak ada akun yang diperlukan
