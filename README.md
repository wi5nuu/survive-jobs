# Surjob AI - #JuaraVibeCoding

Surjob AI adalah platform cerdas yang dirancang untuk memberikan dukungan hukum, analisis situasi kerja, dan panduan kesehatan mental bagi pekerja di Indonesia secara anonim dan aman.

## 🚀 Vision
Memberdayakan pekerja Indonesia untuk memahami hak-hak mereka di tempat kerja dan memberikan solusi taktis untuk menghadapi lingkungan kerja yang tidak sehat melalui bantuan kecerdasan buatan.

## 🎯 Target Audience
Pekerja Indonesia di berbagai sektor (kantoran, pabrik, retail, hingga *freelance*) yang menghadapi tantangan seperti:
- Lingkungan kerja *toxic* atau *gaslighting*.
- Pelanggaran kontrak kerja dan pengupahan (lembur tidak dibayar).
- Kebingungan mengenai proses pengunduran diri (*resign*) yang aman.

## 🛠 Tech Stack
- **AI Engine:** Google Gemini (via Google AI Studio) untuk analisis situasi kerja dan draf respons hukum.
- **Backend:** Node.js dengan Express.js.
- **Frontend:** React (TypeScript) + Vite + Tailwind CSS (Vanilla CSS).
- **Database:** Firebase Firestore (Real-time).
- **Deployment:** Google Cloud Run.
- **Development Tooling:** TSX, Framer Motion (untuk interaksi UI yang dinamis).

## 🛡 Security & Privacy
Keamanan pengguna adalah prioritas utama kami:
- **Anonimitas:** Data curhat pengguna dienkripsi dan disamarkan, tidak ada identitas yang disimpan secara eksplisit.
- **Credential Protection:** Kunci API dikelola secara aman menggunakan variabel lingkungan (`.env`) dan di-inject secara server-side melalui Google Cloud Run.
- **Privacy by Design:** Kami tidak meminta informasi pribadi dalam proses konsultasi.
- **Data Minimization:** Kami hanya menyimpan data curhat yang diperlukan untuk analisis solidaritas komunitas.
